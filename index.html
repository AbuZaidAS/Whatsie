Frequently Asked Questions

    Is Whatsie secure?

    Using Whatsie is as safe and secure as using the official WhatsApp Web app. In fact, Whatsie itself uses WhatsApp Web, so all your messages and other account data are not handled by my app, but by the official WhatsApp Web client.

    Does Whatsie work with iPhones?

    Yes, it does! WhatsApp solved the compatibility problem with iOS devices in August 2015.

    How do I install other languages for the spell checker?

    On OS X and Windows, you're limited by the amount of dictionaries your operating system provides. On Linux, Whatsie uses the dictionaries provided by Hunspell. If you want more languages, install hunspell and the hunspell-* package for the language you want (e.g. hunspell-es for Spanish).

    How does Whatsie work under the hood?

    Whatsie is a native desktop app built on Electron. Basically, it's a wrapper for WhatsApp Web with additional features such as themes and better integration with your operating system.

    Why do I see so many Whatsie processes in Task Manager?

    That's because Whatsie works just like Chrome – it splits itself into several processes, each with their own role. There's one process for the app itself, another is in charge with the notifications, another with the menus, there's one process for each app window, and so on. This means that the OS can run things in parallel to increase their responsiveness, besides other performance benefits. So it's all a good thing!

    What happened to WhatsApp for Desktop?

    Whatsie is the successor of WhatsApp for Desktop, which was scrapped and sold for parts (kidding). I built Whatsie from the ground up using Electron, the more powerful and battery efficient alternative to NW.js, so it's now faster and better than the old app.

    How do I get rid of WhatsApp for Desktop?

    OS X: Move the app into Trash or use AppCleaner which will also delete your preferences, cache and other temporary files.
    Windows: Follow these steps.
    Linux: Use your package manager to remove Whatsie. Also delete this folder: ~/.config/whatsappfordesktop.

    How can I uninstall/remove Whatsie?

    OS X: Move the app into Trash or use AppCleaner which will also delete your preferences, cache and other temporary files.
    Windows (portable): Just delete the app folder. Whatsie didn't create any other files.
    Windows (installer): Go to Control Panel > Change or remove a program, then select Whatsie and click Uninstall.
    Linux: Use your package manager to remove Whatsie.

    Will Whatsie eat my battery or RAM?

    Literally, no. Virtually, yes, just a bit. However, it's much more efficient than the old WhatsApp for Desktop. On my Mac, Whatsie uses around 240MB of RAM and just 3% CPU while idle.
